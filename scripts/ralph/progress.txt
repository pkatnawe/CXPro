- Assignments table uses user_id directly (not participation_id) 
- discipline_scopes table has no updated_at column (only created_at)

- Test functions must use transactions with SET LOCAL for auth.uid() to work

- Assignments table has NO role column - role lives in memberships table (User × Org relationship)
